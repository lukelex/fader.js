fade.js
=======

A micro js to fade in and fade out elements.

Usage
=======

```javascript
fadeOutWithId("elemId", 5);

fadeInWithId("elemId", 3);

fadeOutWithClass("elemClass", 0, 4);

fadeInWithClass("elemClass", 1, 2);
```
